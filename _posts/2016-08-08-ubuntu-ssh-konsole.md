##  Konsole

### 主页

[主页](#https://konsole.kde.org/)

### 特点

- 多标签式终端
- 半透明背景
- 支持拆分视图模式
- 目录和SSH书签化
- 可定制的配色方案
- 可定制的按键绑定
- 终端中的活动通知警告
- 增量搜索
- 支持Dolphin文件管理器
- 支持导出显示为普通文本和HTML格式


### 安装

```bash
sudo apt-get install konsole
```
